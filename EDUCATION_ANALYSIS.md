# 대한민국 영어교육 격차 분석 및 해결 방안

## 1. 핵심 문제 진단

### 1.1 읽기 노출량 격차 (한국 vs 미국)

| 학교급 | 한국 학생 | 미국 학생 | 격차 배율 |
|---|---|---|---|
| 초등학교 (3~6학년) | 32,000~40,000 단어 | 1,500,000 단어 | ~40배 |
| 중학교 (7~9학년) | 60,000~75,000 단어 | 2,250,000 단어 | ~33배 |
| 고등학교 (10~12학년) | 75,000~105,000 단어 | 6,000,000 단어 | ~66배 |
| **합계 (3~12학년)** | **167,000~220,000 단어** | **9,750,000 단어** | **약 44배** |

한국 학생은 미국 학생 대비 약 1/44 수준의 영어 텍스트에만 노출됩니다. 이는 교과서 중심의 제한된 읽기 환경에서 비롯됩니다.

---

### 1.2 수업 시간 부족

| 항목 | 시간 |
|---|---|
| 한국 영어 수업 누적 (3~12학년) | **980시간** |
| CEFR B1 달성 필요 시간 | 1,200~2,000시간 |
| **부족 시간** | **220~1,020시간** |

현재 교육과정으로는 CEFR B1 (중급 하) 수준 달성이 구조적으로 불가능합니다.

---

### 1.3 교육과정-CEFR 정렬 현황

| 학교급 | 학년 | 한국 목표 | CEFR | 실제 도달 |
|---|---|---|---|---|
| 초등 | 3~4 | 기초 의사소통 | A1 | A1 미만 (다수) |
| 초등 | 5~6 | 친숙한 주제 듣기/말하기 | A1~A2 | A1 |
| 중학 | 1 | 일상생활 듣기/읽기 이해 | A2 | A1~A2 |
| 중학 | 2~3 | 친숙한 주제 말하기/쓰기 | A2~B1 | A2 |
| 고등 | 1 | 복합적 이해와 표현 | B1 | A2~B1 |
| 고등 | 2~3 | 추상적 주제 해석/표현 | B1~B2 | B1 미만 (다수) |

**핵심 격차:** 교육과정 목표와 실제 도달 수준 사이에 지속적 갭이 존재합니다.

---

### 1.4 Lexile 수준 분석

| 학년 | 교육과정 목표 Lexile | 수능 요구 수준 | 격차 |
|---|---|---|---|
| 중1 | 200~400L | - | - |
| 중2 | 400~600L | - | - |
| 중3 | 500~700L | - | - |
| 고1 | 700~900L | - | - |
| 고2 | 800~1,000L | - | - |
| 고3 | 900~1,200L | 1,000~1,200L | 실제 학생 다수 800L 이하 |

수능 영어 지문은 Lexile 1,000~1,200L 수준이나, 상당수 고3 학생이 800L 이하에 머물러 있습니다.

## 2. 구조적 원인 분석

### 2.1 문법 중심 교육
- 의사소통 능력보다 문법 지식과 어휘 암기에 편중
- 읽기/듣기는 상대적으로 강하나, 말하기/쓰기는 현저히 부족
- 입시(수능) 중심 평가가 문법·독해 위주 교육을 강화

### 2.2 실제 언어 사용 기회 부족
- 교실 밖 영어 사용 환경이 극히 제한적 (EFL 환경)
- 원어민 교사 접촉 시간 부족
- 실생활 맥락의 언어 활동 부재

### 2.3 평가 방식의 불일치
- 수능은 5지선다 객관식 중심 → 생산적 기능(말하기/쓰기) 미평가
- CEFR은 4기능 통합 평가 → 한국 평가 체계와 괴리
- 수행평가 비중은 증가하나 표준화 부족

### 2.4 교사 지원 부족
- 교사 1인당 관리 학생 수 과다
- 개별화 수업 설계 시간 부족
- 최신 교수법·자료 접근성 제한

## 3. 해결 방안

### 3.1 단기 방안 (1~2년)

#### AI 기반 교사 지원 시스템 활용
- **교육과정 AI 해석기**: 2022 개정 교육과정 성취기준을 실시간으로 해석하고 수업에 연결
- **AI 수업 설계**: 학년·기능·Lexile 수준에 맞는 수업 계획 자동 생성으로 교사 업무 경감
- **AI 문항 생성**: 수능 유형별 맞춤 문항 자동 생성으로 다양한 수준의 평가 자료 확보
- **Lexile 진단 + 교수전략**: 학생별 읽기 수준 측정 후 AI가 맞춤 교수전략 제안

#### 읽기 노출량 확대
- 다독(Extensive Reading) 프로그램 도입: 학생 Lexile 수준에 맞는 자료 추천
- 디지털 리딩 플랫폼 활용으로 교과서 외 텍스트 노출 확대
- 목표: 연간 읽기량 현재 대비 5~10배 증가

### 3.2 중기 방안 (2~3년)

#### CEFR 기반 평가 체계 전환
- 수행평가에 CEFR Can-do 서술문 기반 루브릭 도입
- 말하기·쓰기 평가 비중 확대 (현 10~20% → 30~40%)
- 포트폴리오 평가 도입으로 성장 과정 추적

#### 맞춤형 학습 경로 설계
- Lexile 수준별 학생 그룹핑 및 차별화 교재 제공
- AI가 학생 수준 변화를 추적하여 교사에게 실시간 피드백
- 교육과정 성취기준 도달률 데이터 기반 수업 개선

#### 교사 역량 강화
- AI 코파일럿 활용 교사 연수 프로그램
- 교수법 공유 커뮤니티 (수업 사례 DB 축적)
- 데이터 리터러시 교육: 학생 데이터 해석 및 활용 역량

### 3.3 장기 방안 (3~5년)

#### 교육과정 구조 개편
- 영어 수업 시수 확대 (최소 CEFR B1 달성 가능 수준: 1,200시간 이상)
- 의사소통 중심 교육과정으로 전면 전환
- 교과 융합 영어 수업 (CLIL: Content and Language Integrated Learning) 도입

#### 수능 영어 평가 혁신
- 듣기·읽기 외 말하기·쓰기 영역 추가 (IBT 방식)
- Lexile 수준 기반 적응형 평가 도입 검토
- 절대평가 확대로 과도한 경쟁 완화

#### 영어 몰입 환경 조성
- 학교 내 English Zone 확대 운영
- AI 대화 파트너를 활용한 말하기 연습 기회 확대
- 국제 교류 프로그램 온라인 확장

## 4. 이 앱의 역할

**대한민국 영어교육 개선 시스템**은 위 해결 방안 중 **단기 방안**의 핵심 도구로 설계되었습니다:

| 문제 | 앱의 해결 기능 |
|---|---|
| 교사 수업 설계 시간 부족 | AI 수업 설계 자동 생성 |
| 학생 수준 파악 어려움 | Lexile 진단 + AI 교수전략 |
| 수능 대비 자료 부족 | AI 문항 생성 (7유형, 3난이도) |
| 교육과정 해석 어려움 | AI 교육과정 Q&A |
| 교육 격차 데이터 부재 | 한국-국제 기준 비교 대시보드 |
| 교육 정보 접근성 | RAG 기반 지식 검색 시스템 |
